# Lana Del Rey Porsche 3D Experience

A 3D interactive web experience featuring a customized Porsche 911 GT3 with a "Lana Del Rey" aesthetic. Built with React, Three.js, and Tailwind CSS.

## Features

- Interactive 3D Porsche model
- Custom pink/rose gold color grading
- Atmospheric lighting and particles
- Responsive UI overlay

## Tech Stack

- **React 18**
- **Three.js / React Three Fiber** for 3D rendering
- **Tailwind CSS** for styling
- **Vite** for build tooling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
